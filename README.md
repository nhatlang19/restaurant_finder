### Get a Zomato API Key ###

- Go to the Tomato developer site at https://developers.zomato.com/api, create an account, and generate a new key

### HOW TO RUN ###

- copy .env.default to .env
- edit ZAMATO_CLIENT_KEY
- run `flutter pub get`
